# Quick Start

This page is the short path through the setup — three stages with links to the detailed guides.

## Stage 1 — Device preparation

Get the child's Android device ready before installing the app:

- Remove (or block) the standard Telegram app so it can't be used as a bypass.
- Optionally, configure a device-level parental control tool (e.g. Google Family Link) to restrict app installs and screen time.

See [Android setup](./android.md).

## Stage 2 — Account preparation

Prepare the child's Telegram account from the parent's side:

- Set a cloud password on the child's account so it can't be transferred to another device without your knowledge.
- Open a parent-side session to monitor account activity.
- Review and limit the child's contact list.

See [Child account preparation](./fortgram.md#child-account-preparation).

## Stage 3 — Client setup

Install and configure FortGram on the child's device:

- Install FortGram as the only Telegram client.
- Set a parental PIN — the child can use the app but can't change its settings.
- Configure content filters, allowed contact types, and other in-app controls.

See [FortGram setup](./fortgram.md#installation).

# Быстрый старт

> 🇬🇧 [Read in English ↑](#quick-start)

Короткий маршрут по настройке — три этапа со ссылками на подробные гайды.

## Этап 1 — Подготовка устройства

Подготовить Android-устройство ребёнка до установки приложения:

- Удалить (или заблокировать) стандартный Telegram, чтобы его нельзя было использовать в обход.
- По желанию — настроить систему родительского контроля на уровне устройства (например, Google Family Link): ограничения на установку приложений, экранное время.

См. [Настройка Android](./android.md).

## Этап 2 — Подготовка аккаунта

Подготовить детский Telegram-аккаунт со стороны родителя:

- Поставить облачный пароль на аккаунт ребёнка — чтобы его нельзя было перенести на другое устройство без вашего ведома.
- Открыть родительскую сессию для наблюдения за активностью аккаунта.
- Проверить и ограничить список контактов ребёнка.

См. [Подготовка детского аккаунта](./fortgram.md#подготовка-детского-аккаунта).

## Этап 3 — Настройка клиента

Установить и настроить FortGram на устройстве ребёнка:

- Установить FortGram как единственный Telegram-клиент.
- Задать родительский PIN — ребёнок может пользоваться приложением, но не может менять его настройки.
- Настроить фильтры контента, допустимые типы контактов и другие встроенные элементы управления.

См. [Установка FortGram](./fortgram.md#установка).
