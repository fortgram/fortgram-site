## 🛡️ About the project

**FortGram** is an Android Telegram client for children, with parental control features built into the app itself. It replaces the standard Telegram app and gives parents a safer, more manageable messenger for their kids.

It pairs well with any device-level parental control setup (for example, Google Family Link) but doesn't depend on a specific one. FortGram is a fork of [Telegram X](https://github.com/TGX-Android/Telegram-X) and is developed in a separate repository.

> This repository hosts the **public website** for FortGram ([fgram.org](https://fgram.org)) and the user-facing documentation. The app source code lives elsewhere.

## 🚀 How to set it up

| Stage | What you do |
|-------|-------------|
| **1. Device** | Remove the standard Telegram app; optionally enable a device-level parental control tool. |
| **2. Account** | Set a cloud password on the child's account, open a parent-side session, review contacts. |
| **3. Client** | Install FortGram, set a parental PIN, configure content filters and contact controls. |

## 🛡️ О проекте

**FortGram** — Android-клиент Telegram для детей, в который встроены функции родительского контроля. Он заменяет стандартное приложение Telegram и даёт родителям более безопасный и управляемый мессенджер для ребёнка.

Он хорошо сочетается с любой системой родительского контроля на уровне устройства (например, Google Family Link), но не привязан к какой-то конкретной. FortGram — форк [Telegram X](https://github.com/TGX-Android/Telegram-X), который разрабатывается в отдельном репозитории.

> В этом репозитории живёт **публичный сайт** FortGram ([fgram.org](https://fgram.org)) и пользовательская документация. Исходный код приложения находится отдельно.

## 🚀 Как настроить

| Этап | Что нужно сделать |
|------|-------------------|
| **1. Устройство** | Удалить стандартный Telegram; по желанию — включить родительский контроль на уровне устройства. |
| **2. Аккаунт** | Поставить облачный пароль на аккаунт ребёнка, открыть родительскую сессию, проверить контакты. |
| **3. Клиент** | Установить FortGram, задать родительский PIN, настроить фильтры контента и управление контактами. |

→ Полный гайд: **[Быстрый старт](./docs/quickstart.md#быстрый-старт)**
