FortGram

A Telegram client for children, with built-in parental control features.

Telegram-клиент для детей со встроенными функциями родительского контроля.

What is FortGram?

FortGram is an Android Telegram client for children, with parental control features built into the app itself. It replaces the standard Telegram app and gives parents a safer, more manageable messenger for their kids.

It pairs well with any device-level parental control setup (for example, Google Family Link), but doesn't depend on a specific one.

It is a fork of Telegram X and lives in a separate repository.

Что такое FortGram?

FortGram — Android-клиент Telegram для детей, в который встроены функции родительского контроля. Он заменяет стандартное приложение Telegram и даёт родителям более безопасный и управляемый мессенджер для ребёнка.

Он хорошо сочетается с любой системой родительского контроля на уровне устройства (например, Google Family Link), но не привязан к какой-то конкретной.

Это форк Telegram X, который живёт в отдельном репозитории.

How to set it up

  1. Prepare the device — remove the standard Telegram app; optionally enable a device-level parental control tool.
  2. Prepare the account — set a cloud password on the child's Telegram account, open a parent-side session, review the contact list.
  3. Set up the client — install FortGram, set a parental PIN, configure content filters and contact controls.

Как настроить

  1. Подготовить устройство — удалить стандартный Telegram; по желанию — включить систему родительского контроля на уровне устройства.
  2. Подготовить аккаунт — поставить облачный пароль на аккаунт ребёнка, открыть родительскую сессию, проверить список контактов.
  3. Настроить клиент — установить FortGram, задать родительский PIN, настроить фильтры контента и управление контактами.
